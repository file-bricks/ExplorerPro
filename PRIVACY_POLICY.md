# Privacy Policy

ExplorerPro is a local desktop application. Its core features operate on files and folders selected by the user and do not require a cloud account, hosted backend, telemetry service, or external analytics provider.

## Local Data

ExplorerPro may read filenames, metadata, previews, and file contents when the user opens folders, searches, previews files, checks for duplicates, or runs privacy scans. This processing happens locally on the user's machine.

## No Telemetry

This repository does not include telemetry, analytics tracking, crash-report upload, advertising identifiers, or remote user profiling.

## Repository Hygiene

Do not commit local folder listings, scan results, private documents, logs, build outputs, lock files, `.env` files, credentials, tokens, or generated release artifacts. The `.gitignore` covers the known local artifacts, but contributors remain responsible for reviewing staged changes before publishing.

## Security Reports

Security-sensitive reports should be submitted through GitHub private vulnerability reporting instead of public issues.
