# Store Listing - ExplorerPro

## Deutsch

### Kurzbeschreibung (max. 100 Zeichen)
Lokale Explorer-Suite mit Vorschau, Suche, Datenschutzprüfung und Duplikatsuche.

### Beschreibung (max. 10.000 Zeichen)
ExplorerPro ist eine lokale Desktop-Suite für Menschen, die täglich mit vielen Dateien,
Ordnern, Projekten und Recherchematerial arbeiten. Die App bündelt mehrere typische
Power-User-Workflows in einer Oberfläche: Mehrtab-Dateibrowser, Vorschau,
Datenschutzprüfung, Duplikatsuche, Synchronisationsprofile, App-Launcher,
Prompt-Sammlung und einen leichten Code-Editor.

**Was ExplorerPro kann:**

- Mehrtab-Dateibrowser mit Breadcrumbs und Kontextmenüs
- Vorschau für PDF, Bilder, Textdateien und Quellcode direkt in der App
- Erweiterte Suche mit Filtern für Typ, Größe, Datum und Inhalt
- Hash-basierte Duplikatsuche für große lokale Datenbestände
- Datenschutzprüfung für sensible Dateinamen oder Inhalte
- Lokale Synchronisationsprofile für ausgewählte Ordnerpaare
- Schnellzugriff auf konfigurierte Tools und Prompt-Sammlungen
- Offline-First: Dateien, Indizes und Einstellungen bleiben lokal auf dem Gerät

**Für wen ist ExplorerPro gedacht?**

- Entwickler und Wissensarbeiter mit vielen lokalen Projektordnern
- Nutzer, die Dateien prüfen wollen, ohne ständig externe Viewer zu öffnen
- Menschen, die sensible lokale Daten vor Weitergabe oder KI-Nutzung prüfen möchten
- Power User, die Suche, Vorschau, Launcher und Sync in einem Tool bündeln wollen

**Warum kein Cloud-Dateimanager?**

ExplorerPro ist bewusst lokal aufgebaut. Die App lädt keine privaten Dateilisten,
Dateiinhalte oder Indizes auf einen Server hoch. Das ist besonders wichtig für
Projektordner, Recherchematerial, Dokumente mit Personendaten und lokale
Arbeitsabläufe mit Datenschutzanforderungen.

**Technische Hinweise:**

- Desktop-Anwendung auf Basis von PySide6 (Qt 6)
- Lokale Konfiguration unter `.explorerpro`
- AGPL-Lizenz; Quellcode und Projektdokumentation sind öffentlich einsehbar
- Windows ist die Hauptplattform; macOS und Linux bleiben Source-/Build-Smokes

## English

### Short Description (max. 100 chars)
Local explorer suite with preview, search, privacy checks, and duplicate detection.

### Description (max. 10,000 chars)
ExplorerPro is a local desktop suite for people who work with many files, folders,
projects, and research materials every day. It combines multiple power-user
workflows in one interface: a multi-tab file browser, preview panel, privacy
checks, duplicate finder, synchronization profiles, app launcher, prompt library,
and a lightweight code editor.

**What ExplorerPro does:**

- Multi-tab file browser with breadcrumbs and context menus
- Built-in preview for PDF, images, text files, and source code
- Advanced search with filters for type, size, date, and content
- Hash-based duplicate detection for large local data sets
- Privacy checks for sensitive filenames or file content
- Local sync profiles for selected folder pairs
- Quick access to configured tools and prompt collections
- Offline-first: files, indexes, and settings stay on the local device

**Who is ExplorerPro for?**

- Developers and knowledge workers with many local project folders
- Users who want to inspect files without constantly switching viewers
- People reviewing sensitive local data before sharing or AI usage
- Power users who want search, preview, launcher, and sync in one tool

**Why not a cloud file manager?**

ExplorerPro is intentionally local-first. It does not upload private file lists,
file contents, or indexes to a server. That matters for project folders, research
material, personal documents, and privacy-sensitive workflows.

**Technical notes:**

- Desktop application built with PySide6 (Qt 6)
- Local configuration under `.explorerpro`
- AGPL-licensed with public source code and project docs
- Windows is the main platform; macOS and Linux stay source/build smoke targets
