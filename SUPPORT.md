# Support

## Deutsch

Für Fragen, Fehlerberichte oder Store-Support zu ExplorerPro ist der bevorzugte
Kanal die GitHub-Issue-Seite des Projekts:

https://github.com/file-bricks/ExplorerPro/issues

Bitte keine privaten Dateilisten, vollständigen Pfade, sensiblen Dateinamen,
Dokumentinhalte oder Exportdateien öffentlich posten. Für reproduzierbare Fehler
reichen anonymisierte Schritte, Screenshots ohne Personendaten und die App-Version.

Wenn der Fehler nur im lokalen Explorer-, Vorschau- oder Datenschutz-Workflow
auftritt, ist ein minimales Testverzeichnis mit unkritischen Beispieldateien
hilfreicher als echte Nutzerdaten.

## English

For questions, bug reports, or store support for ExplorerPro, the preferred
channel is the project's GitHub issue page:

https://github.com/file-bricks/ExplorerPro/issues

Do not post private file lists, full paths, sensitive filenames, document
content, or export files publicly. For reproducible bugs, anonymized steps,
screenshots without personal data, and the app version are sufficient.

If the problem only appears in the local explorer, preview, or privacy workflow,
a minimal test folder with non-sensitive sample files is more useful than real
user data.
